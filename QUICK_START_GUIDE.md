# FRAS Report Features - Quick Start Guide

## 📊 Available Reports

### 1. Daily Attendance Report
**What it does**: Shows who was present/absent on a specific date

**How to use**:
1. Click **📅 Daily Report** on the dashboard
2. Select Program, Year, Section, and Date
3. Click **Generate Report**
4. View the attendance table with statistics
5. Use the **Print** button (Ctrl+P) to print

**What you'll see**:
- Present count
- Absent count  
- Total students
- Detailed table with student names and status

---

### 2. Monthly Attendance Percentage
**What it does**: Shows how many classes each student attended during a month

**How to use**:
1. Click **📊 Monthly Report** on the dashboard
2. Select Program, Year, Month, and Calendar Year
3. Click **Generate Report**
4. Review percentage breakdown for each student

**What you'll see**:
- Student name and ID
- Number of days present
- Total days attendance taken
- Attendance percentage (%)
- Visual progress bar
- Color indicators:
  - 🟢 Green: 75%+ (Good)
  - 🟠 Orange: 50-75% (Average)
  - 🔴 Red: <50% (Poor)

---

### 3. Export to CSV/Excel
**What it does**: Download attendance data for external analysis

**How to use**:
1. Click **💾 Export Report** on the dashboard
2. Choose report type:
   - **Daily Report**: For specific section and date range
   - **Monthly Report**: For monthly percentages
   - **Complete Export**: For all attendance records
3. Choose format:
   - **CSV**: For Excel, Google Sheets, data analysis
   - **Excel**: Pre-formatted with colors and styling
4. Fill in the required filters
5. Click **📥 Download Report**

**Excel Features**:
- Professional gradient headers
- Color-coded status (Green for Present, Red for Absent)
- Automatic column width adjustment
- Proper borders and alignment
- Generated timestamp

**CSV Features**:
- Universal compatibility
- Easy import to databases
- Simple, clean format
- Can be opened with any spreadsheet app

## 🎯 Common Tasks

### Check Attendance for a Specific Date
1. Go to **Daily Report**
2. Select the section and date
3. Instant view of who attended

### Find Students with Low Attendance
1. Go to **Monthly Report**
2. Look for rows with percentage < 50%
3. These are marked in red

### Share Attendance Data with Others
1. Go to **Export Report**
2. Choose format (CSV for sharing, Excel for formatting)
3. Download and send the file
4. Recipient can open in any spreadsheet app

### Create Attendance Backup
1. Go to **Export Report**
2. Select "Complete Export"
3. Choose "Excel" format
4. Download - this creates a complete backup of all attendance

### Generate Monthly Report for Multiple Classes
1. Go to **Monthly Report**
2. Select each class combination
3. Download each one
4. Combine in a master spreadsheet

## 📈 Data Interpretation

### Attendance Percentage Calculation
```
Percentage = (Present Days / Total Days) × 100
```

Example:
- Student attended 18 out of 20 classes
- Percentage = (18 / 20) × 100 = 90%

### What Each Status Means
- **Present**: Student marked as present on that date
- **Absent**: Student marked as absent on that date
- **No Record**: No attendance data for that date (student might have a holiday)

## ✅ Tips & Best Practices

1. **Before exporting**:
   - Verify the filters are correct
   - Check the date range
   - Select the right program/year/section

2. **When sharing reports**:
   - Use Excel format for better presentation
   - Use CSV for data import to other systems

3. **For analysis**:
   - Use Complete Export to get all data
   - Import into Excel or Google Sheets
   - Create pivot tables for advanced analysis

4. **Printing**:
   - Use the browser's print function (Ctrl+P)
   - Select "Save as PDF" for archival
   - Adjust margins if needed

5. **Regular monitoring**:
   - Check daily reports at end of day
   - Review monthly reports mid-month
   - Export quarterly for records

## 🔍 Troubleshooting

### No data showing?
- Check if attendance has been marked for that date
- Verify you selected the correct program/year/section
- Try a different date range

### Export file won't open?
- Excel format requires Excel, LibreOffice, or Google Sheets
- CSV format works with any text editor or spreadsheet app
- Try opening with a different application

### Report is blank?
- Make sure students are registered in that section
- Check if attendance was actually marked
- Try a different month/date
